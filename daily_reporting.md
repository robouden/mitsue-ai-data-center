# Daily Reporting System

## Structure

The daily reports now include a section for "Activities by Project" to track what work is done for which projects. This provides better visibility into project activities.

## Daily Report Files
Daily reports are stored in the `memory/` directory with filenames in the format `YYYY-MM-DD.md`.

## Sections
1. Activities by Project - Lists work done for each project
2. Notes - Additional context and information
3. Summary - Overall summary of the day

## Example Structure:

# Daily Report - May 16, 2026

## Activities by Project

### Mitsue Village Project AI Data Center
- [List of activities]

### Other Activities
- [Other work not associated with specific projects]

## Notes
- [Additional context]

## Summary
- [Daily summary]
